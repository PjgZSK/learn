# Register
## 1. register operation
* view   
    > `:reg`  
    > `:reg {register}`

* set and paste  
    > normal mode  
    >> copy : `{register}y`  
    >> paste : `{register}p` 

    > command mode  
    >> paste : `<Ctrl-R>{register}`

reference link: https://harttle.land/2016/07/25/vim-registers.html
