Rasterization overview
1. Clip Space Transformation
    -> clip space(x,y,z,w)
2. Normalized Coodinates
    clip space -> normalized device coordinates
3. Windows Transform
    normalized device coordinates -> window coordinates
4. Scan Conversion
    -> fragment
5. Fragment Processing
    fragment -> color, depth
6. Fragment Writing
    fragment(color, depth) -> destination image

Color
    color description: a series of numbers on the range [0, 1], each
        of the number corresponds to the intensity of a particular reference color
    colorspace: the set of reference colors
    RGB colorspace and CMYK colorspace
