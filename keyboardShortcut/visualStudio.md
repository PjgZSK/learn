# Visual Studio Keyboard Shortcut
## In Common Use
* go to definition  
    `F12`  
* go to declaration  
    `Ctrl+F12`  
* quick view  
    `Alt+F12`  
* find all refercnece  
    `Shift+F12`  
* quick fix  
    `Alt+Enter`  
* rename  
    `Ctrl+R, Ctrl+R`
* quick code comment  
    `Ctrl+/`  
* quick cancel code comment  
    `Ctrl+K, Ctrl+U`  
* view call stucture  
    `Ctrl+K,Ctrl+T`
