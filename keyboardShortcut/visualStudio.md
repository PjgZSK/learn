# Visual Studio Keyboard Shortcut
## In Common Use
* go to definition  
    `F12`  
* go to declaration  
    `Ctrl+F12`  
* quick view  
    `Alt+F12`  
* find all refercnece  
    `Shift+F12`  
* quick fix  
    `Alt+Enter`  
* rename  
    `Ctrl+R, Ctrl+R`
* quick code comment  
    `Ctrl+/`  
* quick cancel code comment  
    `Ctrl+K, Ctrl+U`  
* view call stucture  
    `Ctrl+K,Ctrl+T`  
* open folder  
    `Ctrl+O`  
* select activity windows  
    `Alt+F7`  
* format code  
    `Ctrl+K,Ctrl+F`  
* previous  
    `Ctrl+-`  
* next  
    `Ctrl+Shift+-` 
* fold code  
    `Ctrl+M+M` //fold current code  
    `Ctrl+M+L` //fold all code
